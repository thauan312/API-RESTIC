# API RESTful CRUD com Express

## Instalação

1. Clone o repositório:

   ```
   git clone <URL_DO_REPOSITÓRIO>

   ```

2. Navegue até o diretório do projeto:

   ```
   cd nome-do-repositorio

   ```

3. Instale as dependências

   ```
   npm install

   ```

Para iniciar o servidor, execute:

   ```
   npm start

   ```
